Introduction

Welcome to the Dude Platform.

The Dude Platform is the next generation computing platform that supports all interactions with Dude Solutions clients, partners, and vendors. The platform is exposed as a set of APIs that enable various integrations and applications to interact with Dude Data.

All Platform APIs are exposed as RESTful APIs that exchange JSON-formatted data.

For a complete introduction, list of conventions, and other information please read the Dude Platform Basics.

Dude Platform Basics

Dude Platform Security

DudeAuth is an OAuth2 compliant identity provider service as well as the primary security API for the Dude Platform. All requests to login, authenticate, and even authorization mechanisms are available via DudeAuth.

In order to begin interacting with the Dude Platform you must have a registered application with us.
